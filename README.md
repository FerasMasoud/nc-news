*DISCRIPTION*:  
    THIS PROJECT IS ABOUT NEW APP THAT CONTAINS ARTICLES ABOUT DIFFERENT TOPICS.

*FUNCTIONALITY*:  
    *HOME/MAIN PAGE*:
        1- INCLUDES ALL ARTICLES AVAILABLE.
        2- ARTICLES CAN BE SORTED BY (CREATED-AT) OR (VOTES)
        3- ARTICLES CAN ALSO BE SORTED IN EITHER (ASCENDING) OR (DESCENDING) ORDER.
        4- EACH ARTICLE TITLE CAN BE CLICKED ON AND OPENED ON A SEPERATE PAGE.
        5- EACH TOPIC CAN BE CLICKED ON WHICH FILTERS ONLY RELATED ARTICLES.

    *SINGLE ARTICLE PAGE*:
        1- ABILITY TO VOTE.
        2- ABILITY TO ADD AND POST A COMMENT. FOR USERNAME PLEASE USE 'jessjelly'.
        3- ABILITY TO DELETE COMMENTS WRITTEN BY 'userjelly'.
        4- ABILITY TO SHOW COMMENTS AND HIDE THEM.

*DISCRIPTION*:  

    THIS PROJECT IS ABOUT NEWS APP THAT CONTAINS ARTICLES ABOUT DIFFERENT TOPICS.

*FUNCTIONALITY*:  

    *HOME/MAIN PAGE*:

        1- INCLUDES ALL ARTICLES AVAILABLE.

        2- ARTICLES CAN BE SORTED BY (CREATED-AT) OR (VOTES)

        3- ARTICLES CAN ALSO BE SORTED IN EITHER (ASCENDING) OR (DESCENDING) ORDER.


        4- EACH ARTICLE TITLE CAN BE CLICKED ON AND OPENED ON A SEPERATE PAGE.

        5- EACH TOPIC CAN BE CLICKED ON WHICH FILTERS ONLY RELATED ARTICLES.

    *SINGLE ARTICLE PAGE*:
    
        1- ABILITY TO VOTE.
    
        2- ABILITY TO ADD AND POST A COMMENT. FOR USERNAME PLEASE USE 'jessjelly'.

        3- ABILITY TO DELETE COMMENTS WRITTEN BY user 'jessjelly'.

        4- ABILITY TO SHOW COMMENTS AND HIDE THEM.

        git clone https://github.com/FerasMasoud/news-project.git

    3-TO OPEN THE REPO YOU JUST CLONED, USE THIS COMMAND: 

        code folderName 

        FOR EXAMPLE : code myrepo   

    4-ONCE YOU OPENED THE FOLDER NOW FOLLOW THESE STEPS: 

        -OPEN THE TERMINAL:

            1-FIRST YOU NEED TO BE IN THE nc-news FOLDER, RUN THIS COMMAND: 'cd nc-news'.

            2-AFTER TO INSTALL ALL NEEDED DEPENDENCIES, RUN THIS COMMAND 'npm install'.

            4-YOU SHOULD SEE A PAGE AUTOMICALLY OPENED IF NOT SIMPLY OPEN YOU BROWSER AND GO TO THIS URL: localhost:3000
